# Adidas US Sales Analysis 

## Project Overview

This project analyzes Adidas US sales data to uncover insights about sales performance, profit, and regional trends using Python.

##  Tools & Technologies

* Python 🐍
* Pandas
* Matplotlib
* Jupyter Notebook

##  Key Analysis

* Region-wise total sales
* Product-wise performance
* City-wise sales trends
* Profit analysis
* Data visualization using bar charts, pie charts, and histograms

##  Key Insights

* Identified top-performing regions and products
* Found patterns in sales distribution
* Analyzed relationship between units sold and total sales
